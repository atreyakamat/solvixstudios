# Solvix Studios

Premium, dark-themed, conversion-focused website for Solvix Studios built with Next.js App Router, TypeScript, Tailwind CSS, Framer Motion, and Lucide icons.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Scripts

- `npm run dev` – start dev server
- `npm run lint` – run ESLint
- `npm run build` – production build
- `npm run start` – run production server
